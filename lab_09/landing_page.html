Alarmo

The HEYwire Alarm Clock is a smart alarm system designed for users to actually get out of bed. Using a Raspberry Pi and a thermal sensor, the device detects body heat and only turns off once the user has physically left their bed. It features a gentle PreAlarm phase to gradually wake the user, followed by a louder alarm paired with a color changing smart light to add chaos! The system communicates with an Android alarm app using an MQTT server, allowing perfect harmony between hardware and software. By combining sensing with interactive feedback, the HEYwire Alarm Clock offers an effective and reliable way to start the day.
